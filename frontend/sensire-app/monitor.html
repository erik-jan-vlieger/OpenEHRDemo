🔍 Hoe werkt de monitor?

De monitor pollt elke 4 seconden EHRbase met deze AQL-query. Dat is dezelfde querytaal die elk openEHR-systeem begrijpt — ongeacht de leverancier.

SELECT e/ehr_id/value AS ehr_id, c/uid/value AS composition_id, c/name/value AS formulier, c/context/start_time/value AS tijdstip, c/composer/name AS behandelaar FROM EHR e CONTAINS COMPOSITION c ORDER BY c/context/start_time/value DESC LIMIT 10

Bij een klik op een compositie haalt de monitor de volledige data op via:

GET /ehrbase/rest/openehr/v1/ehr/ {ehr_id}/composition/ {uid}?format=FLAT → Retourneert FLAT JSON → Monitor vertaalt via WebTemplate → Toont: pad | NL label | waarde

Kernboodschap: "Dit is dezelfde data, bevraagd via een internationale querytaal. Geen API van een leverancier — pure openEHR."
